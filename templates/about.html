About This Project

This is Lab 7 — a complete CI/CD pipeline demonstration.

The Stack

  • Flask — lightweight Python web framework
  • Bootstrap 5 — responsive modern UI
  • Docker — containerization
  • Kubernetes — orchestration (Docker Desktop)
  • Jenkins — automated build & deploy
  • GitHub — source of truth

The Flow

  1. Developer pushes code to GitHub.
  2. Jenkins detects the change and starts the pipeline.
  3. Docker image is built and pushed to Docker Hub.
  4. Kubernetes pulls the new image and rolls out the update with zero downtime.

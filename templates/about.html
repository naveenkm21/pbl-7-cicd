About DevOps Toolkit

A small but real-world utility site engineers actually reach for — deployed end-to-end with a CI/CD pipeline.

Tools included

  • JSON Formatter — validate config files, API responses, log lines.
  • Base64 — encode/decode Kubernetes secrets, JWT segments, basic-auth headers.
  • Hash Generator — quickly compute file/text checksums.
  • UUID & Password Generators — safe random IDs and credentials.

CI/CD architecture

  1. GitHub — source of truth.
  2. Jenkins polls SCM, builds the Docker image tagged with the build number.
  3. Image is pushed to Docker Hub.
  4. kubectl apply rolls out the new image across 2 replicas — zero downtime.
  5. Traffic is load-balanced via a NodePort Service.

Why this app fits Kubernetes

Every endpoint is stateless — no database, no session affinity required. Each request is independently handled by whichever pod the Service routes it to. That's why the footer can show different pod names on different requests.
